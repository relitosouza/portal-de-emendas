# Política de Segurança

Esta plataforma é projetada com foco na proteção de dados e integridade do cidadão. Para garantir a segurança contínua do sistema, seguimos as diretrizes abaixo:

## Reportando Vulnerabilidades

Sua contribuição é essencial para a segurança deste projeto. Se você descobrir uma vulnerabilidade de segurança, pedimos que **não abra uma Issue pública**. Em vez disso, relate o problema de forma responsável seguindo o processo abaixo:

1.  Envie um detalhamento para a equipe técnica responsável.
    *   *Nota*: Recomendamos o contato direto via e-mail oficial do suporte administrativo municipal.
2.  Forneça o máximo de informações possível para que possamos reproduzir o problema.
3.  Aguarde a confirmação da equipe antes de qualquer divulgação pública.

Agradecemos sua cooperação para manter este sistema seguro para todos os munícipes de Osasco.
